# philosophers

## Project Overview
The Classic Dining Philosophers problem.

Simulates philosophers eating, sleeping, and thinking under supervision of a Monitor thread.

## Concepts Covered
- Multithreading
- Mutexes
- Race condition prevention
- Timing constraints

## Build
```
make
```

## Usage

```
./philo 5 800 200 200
```
Arguments:
number of philosophers / time to die / time to eat / time to sleep
